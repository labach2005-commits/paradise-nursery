# e-plantShopping

## Project Name
e-plantShopping

## Project Description
e-plantShopping is an online plant shopping application that allows users to browse houseplants, view product details, add plants to a shopping cart, and manage their cart before checkout.

## Features
- Display a list of available plants
- Add plants to the shopping cart
- Increase or decrease item quantity
- Remove items from the cart
- View total cart price

## Technologies Used
- React
- Vite
- JavaScript
- CSS
